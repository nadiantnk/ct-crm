# КТ Диагностика — CRM

Система записи пациентов на КТ-исследования в диагностическом центре.

## Что это

Прототип CRM-системы для сети стоматологических диагностических центров. Позволяет вести запись пациентов, управлять филиалами, клиниками, услугами и формировать отчёты.

## Запуск

Открыть `ct-crm.html` в браузере. Требуется подключение к интернету (React и шрифты загружаются с CDN).

Или через локальный сервер (без ошибок безопасности):

```bash
python -m http.server 8080
```

Затем открыть: `http://localhost:8080/ct-crm.html`

**Логин:** admin  
**Пароль:** admin

## Функциональность

- Расписание по 30-минутным слотам (09:00–20:00)
- Запись пациентов с автоформатированием телефона (+7) и ФИО
- Переключение между филиалами
- Поиск по ФИО и номеру телефона
- Управление клиниками (CRUD, поиск, сортировка по алфавиту)
- Управление услугами по категориям (3D, 2D, пакетные)
- Система скидок с автоматическим расчётом итоговой стоимости
- Цветовые статусы записей (пришёл/оплатил)
- Защита от двойной записи на один слот
- Отчёты по выручке и пациентам (день/месяц, по филиалам)
- Фильтры по дате, клинике, типу исследования

## Стек

- React 18 (CDN)
- Babel (in-browser, только для прототипа)
- Vanilla CSS-in-JS

## Статус

Прототип / MVP. Данные хранятся в оперативной памяти, сбрасываются при перезагрузке.
